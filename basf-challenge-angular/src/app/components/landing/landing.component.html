Challenge Description

This is the challenge application for the full stack position at BASF. The description of the challenge is teh following:

Challenge: Build a Full-Stack Web Application

Your task is to build a full-stack web application that allows users to create, read, update and delete items from a list. The application should consist of two parts: a back-end API and a front-end user interface.

Back-end API

The back-end API should be built using Java and Spring Boot. It should provide the following endpoints:

GET /items: Returns a list of all items.
GET /items/:id: Returns the item with the specified ID.
POST /items: Creates a new item.
PUT /items/:id: Updates an existing item with the specified ID.
DELETE /items/:id: Deletes the item with the specified ID.
Each item should have the following attributes:

id: A unique identifier for the item.
name: The name of the item.
description: A description of the item.
The API should store data in a PostgreSQL database. Use Docker Compose to spin up the database instance.

Front-end User Interface

The front-end user interface should be built using Angular and TypeScript. It should allow users to:

View a list of all items.
View the details of a specific item.
Create a new item.
Update an existing item.
Delete an existing item.
The user interface should use Bootstrap or Material Design for styling.

Deliverables

Your solution should include:

Source code for the back-end API and front-end user interface.
A Docker Compose file that spins up the entire application (API, database, and user interface).
Instructions on how to build and run the application.
